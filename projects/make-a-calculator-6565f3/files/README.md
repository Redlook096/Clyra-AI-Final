# Calculator

A premium, production-ready calculator built with React 19, TypeScript, and Vite.

## Features

- **Arithmetic Operations:** Add, subtract, multiply, divide
- **Keyboard Support:** Full keyboard input with mapped keys
- **Calculation History:** Persistent history with local storage
- **Live Preview:** See result as you type
- **Error Handling:** Division by zero, invalid expressions
- **Responsive:** Works on desktop and mobile
- **Dark Theme:** Premium dark mode design

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0-9` | Append digit |
| `.` | Decimal point |
| `+ - * /` | Operators |
| `Enter` / `=` | Evaluate |
| `Backspace` | Delete last character |
| `Escape` / `Delete` | Clear all |

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- React 19
- TypeScript
- Vite 6
- CSS (no external UI libraries)
