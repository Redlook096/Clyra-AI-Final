# Todo List App

## Goal
Build a simple, visually appealing to-do list application using React with TypeScript. The app allows users to add, delete, and mark tasks as complete.

## Requirements
- Add a new todo item via input and submit button.
- Display list of todos with checkbox to mark as complete.
- Delete todos with a delete button.
- Clear input after adding.
- Show count of remaining todos.
- Premium minimal UI with glassy effects and smooth transitions.

## Execution Gates
1. Setup React project structure (App.tsx, styles.css).
2. Implement state management for todos array.
3. Build UI components: input form, todo list, individual todo item.
4. Style with glassmorphism, rounded corners, backdrop blur, subtle shadows, dark theme.
5. Test logic: add, complete, delete, remaining count.

## Notes
- Use functional components and hooks (useState).
- No external libraries needed.
- Focus on polished UI/UX.
