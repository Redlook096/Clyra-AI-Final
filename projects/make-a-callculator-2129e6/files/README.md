# Callculator

A premium dark-mode calculator with a glass-morphism design, built with React + Vite + TypeScript.

## Features

- Basic arithmetic: +, -, ×, ÷
- Utility operations: AC, ±, %, ⌫
- Decimal input with smart guard
- Session calculation history (last 5 results)
- Full keyboard support
- Premium glass-morphism dark UI
- Smooth animations and haptic-like feedback
- Fully responsive (mobile + desktop)

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| 0-9 | Input digit |
| . | Decimal point |
| + | Add |
| - | Subtract |
| * | Multiply |
| / | Divide |
| Enter / = | Calculate |
| Escape | Clear (AC) |
| Backspace | Delete last digit |
