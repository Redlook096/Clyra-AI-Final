## Goal
Create a premium landing page for Google, replicating the classic search homepage with a modern glassmorphism aesthetic. The page should be fully responsive, interactive, and visually polished.

## Requirements
- Display the Google logo prominently at the top.
- Search bar with glassmorphism effect (backdrop blur, semi-transparent background, rounded corners).
- Placeholder text: "Search Google or type a URL".
- Two buttons: "Google Search" and "I'm Feeling Lucky", with glassy styling and hover effects.
- Centered layout, vertically and horizontally.
- Subtle animated gradient background to enhance the glass effect.
- Clean, minimal design with smooth transitions.

## Execution Gates
1. Initialize React project (if needed) and configure styles.
2. Build the UI layout: logo, search input, buttons.
3. Implement glassmorphism CSS using backdrop-filter, rgba colors, and box-shadow.
4. Add responsive design (mobile-friendly).
5. Polish with animations and interactions (focus state, button hover).

## Technical Notes
- Use React functional component with hooks.
- All styles in a single CSS file (no CSS-in-JS).
- No external dependencies.
- The logo can be a styled text or inline SVG (here we use a simple SVG path for Google's colors).
