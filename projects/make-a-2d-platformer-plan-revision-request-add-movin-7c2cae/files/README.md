# Skybound Runner

A polished browser 2D platformer generated through Vibe Coder Plan Mode.

## Controls

- Arrow keys or A/D to move
- Space, W, or Arrow Up to jump
- P to pause
- R to restart

## Features

- Canvas-based platforming
- Coins, hazards, and portal goal
- Win/lost/pause states
- Responsive premium game shell

## Run

```bash
npm run dev
npm run build
```
