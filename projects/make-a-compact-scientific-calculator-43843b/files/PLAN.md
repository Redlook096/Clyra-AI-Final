# Plan

Build a compact scientific calculator as a single HTML file with inline CSS/JS.

**Features:**
- Basic operations: +, −, ×, ÷
- Scientific functions: sin, cos, tan, log, ln, sqrt, power, factorial
- Parentheses, memory (MC/MR/M+/M−), degree/radian toggle
- Clean, compact, responsive UI with dark theme

**Steps:**
1. Create `index.html` with HTML skeleton + CSS layout
2. Add JavaScript calc engine + UI event bindings
3. Verify by opening in a browser
