## Goal
Build a premium, minimally designed calculator with glassmorphism effects. The calculator should support basic arithmetic operations (+, -, ×, ÷) and provide a polished, modern user experience.

## Requirements
- Display current input and result.
- Buttons for digits 0-9, decimal point, operators, clear (C), equals (=).
- Glassmorphism aesthetic: translucent backgrounds, backdrop blur, subtle shadows, rounded corners.
- Responsive layout, works on mobile and desktop.
- Clean state management using React hooks.

## Execution Gates
1. **Gate 1: Design & Structure** - Define component layout, button grid, display area.
2. **Gate 2: Core Logic** - Implement calculator state (current, previous, operator), handle button clicks, calculate results.
3. **Gate 3: Styling** - Apply glassmorphism CSS: background blur, frosted glass effect, smooth animations.
4. **Gate 4: Polish & Test** - Ensure error handling (division by zero), edge cases, responsive design.
