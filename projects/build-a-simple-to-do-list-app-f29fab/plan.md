# Simple To-Do List App

## Goal
Build a premium, minimal to-do list application with glassmorphism design, allowing users to add, complete, and delete tasks.

## Requirements
- Add new todos via an input field and a button
- Mark todos as completed with a checkbox
- Delete todos with a delete button
- Glassy, frosted glass UI with subtle shadows and blur effects
- Responsive and polished layout

## Execution Gates
1. **Gate 1**: Set up React project structure with `App.tsx` and `styles.css`.
2. **Gate 2**: Implement `TodoItem` component with checkbox and delete button.
3. **Gate 3**: Implement `TodoInput` component for adding tasks.
4. **Gate 4**: Integrate state management in `App.tsx` using `useState`.
5. **Gate 5**: Apply glassmorphism styling and final polish.

## Future Enhancements
- Persist todos in local storage
- Add drag-and-drop reordering
- Categorize tasks
